# Cancer-Prediction-System_ANN_MPOnline

Dataset :Breast Cancer Wisconsin (Diagnostic) Dataset

Link: https://archive.ics.uci.edu/ml/datasets/Breast+Cancer+Wisconsin+(Diagnostic)


Test Accuracy: 0.9737 (111/114 correct)

The training loss starts at ~0.40 and plummets to almost exactly 0.0000 by epoch 30, remaining flat until epoch 50.
